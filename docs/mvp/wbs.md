### クリティカルパスと進め方のポイント
- クリティカルパスは **1.1 → 3.1 → 5.1 → 5.4〜5.7 → 7.2**（実装の縦のライン）
- **§4 データ整備は実装と完全に並行できる**うえ最大の工数なので、実装に疲れた日の作業として細切れに消化するのが現実的
- 最初の2週間で §1〜§3 と 5.1〜5.3 まで到達できれば順調
- 工数を削りたい場合の優先削減候補：6.2（構造化データ）→ 5.5 の絞り込み機能（一覧表示のみで先行公開）
